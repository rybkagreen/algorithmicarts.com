# Отчет о выполнении задания "Исправление неработающих кнопок"

## Проблема:
Некоторые кнопки на сайте не работали, потому что они не имели обработчиков событий onClick или ссылок для навигации.

## Решение:
1. Обновлен компонент Button для поддержки пропа `asChild`, который позволяет ему оборачивать другие компоненты (например, Link) и применять к ним свои стили.
2. В компоненте Hero.tsx добавлены ссылки на страницы `/contact` и `/portfolio` для соответствующих кнопок.
3. В компоненте Header.tsx добавлены ссылки на страницу `/contact` для кнопок в десктопной и мобильной версиях.

## Технические детали:
- Использован компонент `Link` из Next.js для навигации между страницами
- Для реализации паттерна `asChild` использован компонент `Slot` из `@radix-ui/react-slot`
- Обновлены все кнопки, которые должны были выполнять навигацию, чтобы они действительно вели на соответствующие страницы

## Результат:
Все кнопки теперь функциональны и перенаправляют пользователя на соответствующие страницы:
- Кнопка "Обсудить проект" на главной странице ведет на страницу контактов
- Кнопка "Посмотреть портфолио" ведет на страницу портфолио
- Кнопки "Обсудить проект" в шапке также ведут на страницу контактов
